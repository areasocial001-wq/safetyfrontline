# Sound Effects Directory

This directory contains sound effect files used for UI animations.

## Required Audio Files

Place the following audio files in this directory:

1. **whoosh.mp3** - Deep spaceship whoosh sound
2. **pop.mp3** - Mechanical pop/click sound
3. **sparkle.mp3** - Energy charge/sparkle sound
4. **impact.mp3** - Heavy bass impact (like Doom door opening)
5. **rise.mp3** - Ascending engine startup sound

## Recommended Sources for Free Professional Sound Effects

### 1. Freesound.org (https://freesound.org/)
- Search for: "sci-fi whoosh", "mechanical pop", "impact bass", "engine startup"
- License: Creative Commons (check individual licenses)
- High quality professional sounds

### 2. ZapSplat (https://www.zapsplat.com/)
- Categories: Sci-Fi, Industrial, Impact
- License: Free for commercial use (with attribution)
- Excellent quality sound effects library

### 3. Mixkit (https://mixkit.co/free-sound-effects/)
- Categories: Sci-Fi, Game, UI Sounds
- License: Free for commercial use
- No attribution required

### 4. Pixabay Sound Effects (https://pixabay.com/sound-effects/)
- Search for: "whoosh", "impact", "sci-fi"
- License: Free for commercial use
- No attribution required

### 5. BBC Sound Effects (https://sound-effects.bbcrewind.co.uk/)
- Search for: Industrial, Sci-Fi categories
- License: RemArc License (free for personal/educational use)
- Professional BBC sound library

## Recommended Sound Characteristics

- **Format**: MP3 or WAV
- **Duration**: 0.1 - 0.8 seconds
- **Sample Rate**: 44.1kHz or 48kHz
- **Bit Depth**: 16-bit minimum
- **Style**: Industrial, sci-fi, mechanical, Doom-inspired
- **Frequencies**: Deep bass (40-250 Hz) for impacts, mid-range for effects
